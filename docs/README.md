# Tic-Tac-Toe — GitHub Pages Version

This package is ready for GitHub Pages static hosting.

## Deploy

1. Upload all files in this folder to your repository.
2. In GitHub, go to **Settings → Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Select your branch and choose either `/ (root)` or `/docs`.
5. Save.

This version has no CDN dependency, no Node server dependency, and uses relative asset paths so it works from a GitHub Pages repo URL like `https://username.github.io/repo-name/`.

Note: GitHub Pages cannot run a Socket.IO/Express multiplayer backend. This is the static Pages-compatible gameplay version that preserves the same app screens and game flow.
